## 实验2
需要安装dart环境，`dart`版本为`2.12.0`

```sh
Dart SDK version: 2.12.0 (stable) (Thu Feb 25 19:50:53 2021 +0100) on "windows_x64"
```
- `lab2.dart`是实验2源码
`ext.dart`是`lab2.dart`的工具类
`pubspec.yaml` 文件 限制了dart支持的语法特性，关闭了dart null safty 检查，否则编译不通过

- **lab2.dart** 可以直接运行
